What is HTML?

  • HTML stands for Hyper Text Markup Language
  • Describes the structure of a web page
  • Consists of a series of elements
  • Elements tell the browser how to display the content
  • Elements are represented by tags

    • HTML Basic Structure

    Basic HTML structure have five tags

    • <!DOCTYPE html>
    • <html>
    • <head>
    • <title>
    • <body>

    HTML Tags

    HTML tags are element names surrounded by angle brackets

    <tagname>content goes here...</tagname>

    Tags Categories

    Basic HTML

    Include the basic html structure and other tags as:

    • <h1> to <h6> - Defines HTML headings.
    • <p> - Defines a paragraph.
    • <br> - Inserts a single line break.
    • <hr> - Defines a thematic change in the content.
    • <!-- --> - Defines a comment.

    Formatting

    • <b> - Bold text
    • <strong> - Important text
    • <i> - Italic text
    • <em> - Emphasized text
    • <mark> - Marked text
    • <small> - Smaller text
    • <del> - Deleted text
    • <ins> - Inserted text
    • <sub> - Subscript text
    • <sup> - Superscript text

    Forms and Inputs

    The <form> element is used to create an user input

    The <input> element has many types:

    • <input type="text"> Show a single line text input field
    • <input type="radio"> Show a radio button
    • <input type="checkbox"> Show a checkbox
    • <input type="submit"> Show a submit button
    • <input type="button"> Show a clickable button

    Images

    • <img> tag is used to embed an image and has two attributes:
      • src - Specifeis the path to the imagem
      • alt - Specifeis an alternate text for the imagem

    Links

    A HTML hyperlink is defined by <a> and has a href attribute to the hyperlink value

    Lists

    HTML has two types of lists:

    • Unordered list
    • starts with <ul> tag
    • Each list item starts with <li> tag

    and

    1. Ordered list
    2. Starts with <ol> tag
    3. Each list item starts with <li>

    Styles and Semantics

    HTML style attribute is used to styles to an element, such as color, font, size, and more
